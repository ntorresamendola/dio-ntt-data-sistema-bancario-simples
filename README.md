
# Criando um Sistema Bancário com Python básico

Projeto parte do Bootcamp "NTT DATA - Engenharia de Dados com Python", oferecido pela DIO. É um programa que simula um sistema bancário simples, com apenas um usuário, com operações bancárias básicas.

Rodando o programa:

```
git clone https://github.com/ntorresamendola/dio-ntt-data-sistema-bancario-simples
python3 app.python
```
Basta seguir as instruções do menu.
